# Shorelark

Simulation of life & evolution - powered by neural networks, genetic algorithms, and high-school math.

**Blog post with details coming!**

# Building

```bash
$ cd libs/simulation-wasm
$ wasm-pack build --release
$ cd ../../www
$ npm install
$ npm run start
```
